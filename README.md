## 👩‍🏫 Como rodar localmente

- Clonando Projeto
  - `git clone https://github.com/cs-matheus-antonino/crud-angular.git`;
  - `cd crud-angular`;
- Rodando Front-end
  - `npm install`;
  - `npm start`;
- Rodando Back-end
  - `npm install -g json-server`;
  - `json-server --watch db.json --port 3001`;
